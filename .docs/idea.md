# dir-tree-maker アイデア

## 作りたいもの
ブラウザ上で入力したら、リアルタイムでディレクトリツリーが表示されるようなサービス

## 差別化のコアコンセプト
設計ドキュメント作成ツールとしての独自ポジションを目指す

### 主要機能
1. **ディレクトリツリー作成** - ブラウザ上で入力したら、リアルタイムでディレクトリツリーが表示される
2. **コメント付き出力** — 各ファイル/ディレクトリにコメントを付けて、READMEや設計書にそのまま貼れる形で出力
3. **テンプレート機能** — Next.js / Rails / FastAPI などのフレームワーク雛形をワンクリックで読み込み
4. **ローカルディレクトリ読み込み** — File System Access API でローカルフォルダを開いてツリー表示
